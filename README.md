# Vibro-Diagnostics

Веб-приложение для учета и анализа данных вибродиагностики промышленного оборудования.

## Обзор

Это приложение, созданное на основе фреймворка **Flask**, представляет собой комплексное решение для инженеров и технических специалистов, занимающихся вибродиагностикой. Оно позволяет централизованно хранить и обрабатывать протоколы измерений, вести базу данных оборудования, сотрудников и объектов, а также генерировать отчеты в формате PDF.

## Технологический стек

*   **Бэкенд:** Python 3.9+ / Flask
*   **База данных:** PostgreSQL
*   **ORM:** SQLAlchemy / Flask-SQLAlchemy
*   **Генерация PDF:** wkhtmltopdf

## Установка и запуск

### 1. Системные зависимости

Перед первым запуском убедитесь, что в вашей системе установлены:

*   **Python (версия 3.9 или выше):** [python.org](https://www.python.org/)
*   **PostgreSQL:** [postgresql.org](https://www.postgresql.org/)
*   **wkhtmltopdf:** [wkhtmltopdf.org](https://wkhtmltopdf.org/downloads.html)

> **Важно:** При установке `wkhtmltopdf` убедитесь, что его исполняемый файл добавлен в системную переменную `PATH`, или укажите путь к нему в переменной окружения `WKHTMLTOPDF_PATH`.

### 2. Настройка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Rail81/vibro-diag.git
    cd vibro-diag
    ```

2.  **Настройте переменные окружения:**

    Создайте файл `.env` в корне проекта. В нем нужно указать параметры подключения к вашей базе данных.

    ```
    SECRET_KEY='your-super-secret-key'
    DATABASE_URL='postgresql://user:password@host:port/dbname'
    WKHTMLTOPDF_PATH='C:\Program Files\wkhtmltopdf\bin\wkhtmltopdf.exe' # Опционально
    ```

3.  **Инициализируйте базу данных:**

    Перед первым запуском необходимо создать таблицы в базе данных.
    ```bash
    python init_db.py
    ```

4.  **Запустите приложение:**

    Просто запустите файл `start.bat`.
    ```bash
    start.bat
    ```
    Скрипт автоматически создаст виртуальное окружение, установит все зависимости и запустит приложение.

После запуска приложение будет доступно по адресу `http://127.0.0.1:5000`.
