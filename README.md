
# Vibro-Diagnostics

Веб-приложение для учета и анализа данных вибродиагностики промышленного оборудования.

## Обзор

Это приложение, созданное на основе фреймворка **Flask**, представляет собой комплексное решение для инженеров и технических специалистов, занимающихся вибродиагностикой. Оно позволяет централизованно хранить и обрабатывать протоколы измерений, вести базу данных оборудования, сотрудников и объектов, а также генерировать отчеты в формате PDF.

### Ключевые возможности

*   **Управление протоколами:** Создание, просмотр, удаление и печать протоколов вибродиагностических обследований.
*   **База данных:** Ведение справочников по оборудованию, объектам, организациям и сотрудникам.
*   **Генерация PDF:** Автоматическое формирование протоколов в формате PDF для архивирования или печати.
*   **Гибкая настройка:** Возможность конфигурации через переменные окружения.

## Технологический стек

*   **Бэкенд:** Python 3.9+ / Flask
*   **База данных:** PostgreSQL
*   **ORM:** SQLAlchemy / Flask-SQLAlchemy
*   **Миграции:** Alembic / Flask-Migrate
*   **Генерация PDF:** wkhtmltopdf

## Установка и запуск

### 1. Системные зависимости

Перед установкой приложения убедитесь, что в вашей системе установлены:

*   **Python (версия 3.9 или выше):** [python.org](https://www.python.org/)
*   **PostgreSQL:** [postgresql.org](https://www.postgresql.org/)
*   **wkhtmltopdf:** [wkhtmltopdf.org](https://wkhtmltopdf.org/downloads.html)

> **Важно:** При установке `wkhtmltopdf` убедитесь, что его исполняемый файл добавлен в системную переменную `PATH`, или укажите путь к нему в переменной окружения `WKHTMLTOPDF_PATH`.

### 2. Настройка проекта

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Rail81/vibro-diag.git
    cd vibro-diag
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Linux/macOS
    venv\Scripts\activate    # Для Windows
    ```

3.  **Установите зависимости Python:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Настройте переменные окружения:**

    Создайте файл `.env` в корне проекта и укажите в нем параметры подключения к вашей базе данных. Вы можете использовать файл `.env.example` как шаблон.

    ```
    SECRET_KEY='your-super-secret-key'
    DATABASE_URL='postgresql://user:password@host:port/dbname'
    WKHTMLTOPDF_PATH='C:\Program Files\wkhtmltopdf\bin\wkhtmltopdf.exe' # Опционально, если не в PATH
    ```

### 3. Инициализация базы данных

1.  **Создайте базу данных** в PostgreSQL с именем, которое вы указали в `DATABASE_URL`.
2.  **Выполните инициализацию:**
    ```bash
    python init_db.py
    ```
    Эта команда создаст все необходимые таблицы и заполнит их начальными данными.

### 4. Запуск приложения

```bash
flask run
```

После запуска приложение будет доступно по адресу `http://127.0.0.1:5000`.
