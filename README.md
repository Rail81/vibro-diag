# Vibro-Diagnostics

Веб-приложение для учета и анализа данных вибродиагностики промышленного оборудования.

## Обзор

Это приложение, созданное на основе фреймворка **Flask**, представляет собой комплексное решение для инженеров и технических специалистов. Оно позволяет централизованно хранить и обрабатывать протоколы измерений, вести базу данных оборудования, сотрудников и объектов, а также генерировать отчеты в формате PDF.

## Установка и запуск

### Шаг 1: Системные зависимости

Перед первым запуском убедитесь, что в вашей системе установлены:

*   **Python (версия 3.9 или выше):** [python.org](https://www.python.org/)
*   **PostgreSQL:** [postgresql.org](https://www.postgresql.org/)
*   **wkhtmltopdf:** [wkhtmltopdf.org](https://wkhtmltopdf.org/downloads.html)

### Шаг 2: Клонирование репозитория

```bash
git clone https://github.com/Rail81/vibro-diag.git
cd vibro-diag
```

### Шаг 3: Конфигурация

Запустите скрипт `configure.bat`. Он задаст вам несколько вопросов и автоматически создаст файл конфигурации `.env`.

```bash
configure.bat
```

### Шаг 4: Настройка базы данных

1.  Убедитесь, что в PostgreSQL **существует база данных**, имя которой вы указали при конфигурации.
2.  Запустите скрипт для создания таблиц:
    ```bash
    python init_db.py
    ```

### Шаг 5: Запуск приложения

Просто запустите `start.bat`. Этот скрипт сам создаст виртуальное окружение, установит все зависимости и запустит веб-сервер.

```bash
start.bat
```

После запуска приложение будет доступно по адресу `http://127.0.0.1:5000`.
