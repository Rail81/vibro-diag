# Vibro-Diagnostics

Веб-приложение для учета и анализа данных вибродиагностики промышленного оборудования.

## Установка и запуск

### Шаг 1: Системные зависимости

Убедитесь, что у вас установлены **Python 3.9+**, **PostgreSQL** и **wkhtmltopdf**.

### Шаг 2: Клонирование репозитория

```bash
git clone https://github.com/Rail81/vibro-diag.git
cd vibro-diag
```

### Шаг 3: Конфигурация

Запустите `configure.bat`, чтобы создать файл `.env` с настройками базы данных.

```bash
configure.bat
```

### Шаг 4: Установка

Запустите `setup.bat`. Он создаст виртуальное окружение и установит все зависимости.

```bash
setup.bat
```

> **Примечание:** Этот шаг нужно выполнить только один раз. Для последующих запусков используйте `run.bat`.

### Шаг 5: Инициализация БД

Перед самым первым запуском создайте таблицы в базе данных:

```bash
python init_db.py
```

### Шаг 6: Ежедневный запуск

Используйте `run.bat` для запуска приложения.

```bash
run.bat
```

Приложение будет доступно по адресу `http://127.0.0.1:5000`.
